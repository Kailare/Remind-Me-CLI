# Remind Me CLI

Put your rep where your mouth is.

## Install

```bash
npm install -g remindme-cli
```

## Quick Start

```bash
# Initialize
reme init

# Log a prediction
reme log YES https://polymarket.com/event/fed-rate-cut

# Check your stats
reme stats

# Check for resolutions
reme check
```

## Why

- Track Polymarket calls locally
- Prove your calls with receipts
- Track accuracy and streaks
- Optional leaderboard submission later
